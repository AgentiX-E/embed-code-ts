# @agentix-e/embed-code-cli

Command-line interface for nomic-embed-code.

## Install

```bash
npm install -g @agentix-e/embed-code-cli
```

## Commands

```bash
embed-code setup              # Download model
embed-code info               # Show model metadata
embed-code embed "code here"  # Generate embeddings
```

Full API reference: [embed-code-ts API Docs](https://agentix-e.github.io/embed-code-ts/api/)
